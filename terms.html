02 / The service

What VULNRA Is

VULNRA is a SaaS platform that automatically tests LLM API endpoints for security vulnerabilities including prompt injection, jailbreaks, encoding bypasses, data exfiltration, and system prompt leakage. It generates compliance-mapped audit reports referencing EU AI Act, India DPDP, NIST AI RMF, OWASP LLM Top 10, and MITRE ATLAS frameworks.

VULNRA is a security testing tool, not a guarantee of security. A passing score means your system passed the probe suite at that point in time — not that it is free of all vulnerabilities. Security posture changes with every model update and configuration change.

04 / Authorised use

You May Only Scan What You Own or Are Authorised to Test

⚠ You may only submit URLs for scanning if you are the owner of that system, or have explicit written authorisation from the owner to conduct security testing against it. Scanning without authorisation is illegal in most jurisdictions and will result in immediate account termination and potential legal referral.

Authorised use includes:

  • Scanning your own LLM API endpoints that you own and operate
  • Scanning endpoints operated by your employer where your role includes security testing responsibilities
  • Scanning client endpoints where you have a signed penetration testing authorisation agreement
  • Scanning public test endpoints explicitly designated for security research (e.g., dedicated bug bounty targets)
05 / Prohibited conduct

What You Must Not Do

The following are strictly prohibited and will result in immediate account termination:

  • Scanning any endpoint without the owner's explicit written authorisation
  • Submitting private IP ranges, localhost, or cloud metadata endpoints as scan targets — these are automatically blocked
  • Using VULNRA to develop attack tools or adversarial datasets for use against third parties
  • Sharing your API key or account credentials with third parties
  • Reverse-engineering, decompiling, or extracting VULNRA's proprietary probe library, scoring logic, or PDF generator
  • Circumventing rate limits via multiple accounts, proxies, or automated account creation
  • Using VULNRA for any activity that violates the IT Act 2000 (India), CFAA (USA), Computer Misuse Act (UK), or equivalent law in your jurisdiction
  • Uploading or transmitting malware or code designed to damage VULNRA's infrastructure
