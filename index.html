Execution Flow

From target endpoint to audit report
in four phases

Deploy VULNRA against any LLM interface. The engine operates autonomously — no complex configuration or manual prompt engineering required.

01
Initialize
Input any REST endpoint URL — whether it's an OpenAI-compatible interface, custom model deployment, or internal proprietary API. VULNRA automatically maps the schema.
02
Execute Probes
The core engine fires 100+ adversarial probe vectors asynchronously — including complex DAN jailbreaks, indirect injection chains, encoding permutations, and structured PII lures.
03
Compute Score
Every successful bypass is assigned a deterministic 0–10 threat score weighted by vulnerability severity, ease of exploitation, and specific regulatory exposure.
04
Generate Audit
Export a cryptographically verifiable PDF audit report documenting all active findings, specific compliance framework violations, and actionable remediation steps.

terminal
# Install the CLI package
$ pip install vulnra
$ vulnra init
 
# Execute unauthenticated scan on target endpoint
$ vulnra scan https://api.internal/v1/chat

[PASS] Jailbreak (DAN_v11) mitigated

[WARN] PII reflection risk identified — Score: 5.7
