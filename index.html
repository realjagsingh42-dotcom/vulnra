Execution Flow

From target to audit report
in four phases

Deploy VULNRA against any LLM interface. The engine operates autonomously — no complex configuration required.

01
Initialize
Input any REST endpoint URL — whether OpenAI-compatible, custom model deployment, or proprietary API. VULNRA automatically maps the schema.
02
Execute Probes
The engine fires 100+ adversarial probe vectors asynchronously — DAN jailbreaks, indirect injection chains, encoding permutations, and structured PII lures.
03
Compute Score
Every bypass is assigned a deterministic 0–10 threat score weighted by severity, ease of exploitation, and specific regulatory exposure.
04
Generate Audit
Export a verifiable PDF audit report documenting all active findings, compliance violations, and actionable remediation steps.
